# KanBan
Lightweight Kanban task board with a React + TypeScript frontend and an Express + MongoDB backend.

## Setup

1) Install dependencies:
```
npm install
```

2) Configure environment:
- `client/.env.example` -> `client/.env`
- `server/.env.example` -> `server/.env`

3) Run the API:
```
npm run dev:server
```

4) Run the frontend:
```
npm run dev
```
